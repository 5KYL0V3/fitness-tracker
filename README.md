# Fitness Tracker

一个轻量级的健身训练记录网页工具，主要用于记录每次训练的动作、组数、重量、次数、RIR/RPE、休息时间、训练状态和备注，方便后续复盘训练表现。

## 项目简介

Fitness Tracker 是一个面向个人使用的训练记录工具。
它适合在手机浏览器中使用，训练时可以快速记录每个动作和每一组的数据，减少纸笔记录或聊天记录带来的混乱。

项目主要目标：

* 记录每次训练的基本信息
* 记录动作、组数、重量、次数等训练数据
* 记录 RIR / RPE、稳定性、发力感、动作感受
* 记录睡眠、饮食、恢复、出汗、训练环境等状态
* 帮助后续进行训练复盘和计划调整

## 功能特点

* 支持按训练部位记录训练
* 支持添加多个训练动作
* 支持记录每个动作的多组数据
* 支持重量、次数、RIR / RPE 等关键训练指标
* 支持训练状态标签
* 支持训练备注
* 支持动作折叠，减少页面混乱
* 支持移动端浏览器使用
* 适合个人训练复盘

## 使用场景

这个项目适合以下情况：

* 健身房训练时快速记录数据
* 训练后整理当天表现
* 观察动作重量和次数变化
* 记录训练状态对表现的影响
* 把训练数据整理后交给 AI 分析

## 本地运行

直接用浏览器打开 `index.html` 即可运行。

也可以在 VSCode 中使用 Live Server 插件运行：

1. 用 VSCode 打开项目文件夹
2. 安装 Live Server 插件
3. 右键 `index.html`
4. 选择 `Open with Live Server`
